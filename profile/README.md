# ToxMCP

**ToxMCP** is a suite of **guardrailed, auditable agentic workflows** for computational toxicology delivered through the **Model Context Protocol (MCP)**.

Start here: https://github.com/ToxMCP/toxmcp

## Showcase module: EPA CompTox MCP

If you want the most “canonical toxicology data” entry point, check out **CompTox MCP**:
- Repo: https://github.com/ToxMCP/comptox-mcp
- Note: requires an EPA CompTox API key (`CTX_API_KEY`)

## Modules

- AOP MCP: https://github.com/ToxMCP/aop-mcp
- ADMETlab MCP: https://github.com/ToxMCP/admetlab-mcp
- O-QT MCP (OECD QSAR Toolbox): https://github.com/ToxMCP/oqt-mcp
- PBPK MCP: https://github.com/ToxMCP/pbpk-mcp
